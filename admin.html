Режим оплати

Коли клієнт оплачує?
Одразу при замовленні
Клієнт обирає страви → обирає провайдера → оплачує → замовлення йде в касу. Рекомендовано для доставки.
Після підтвердження закладом
Клієнт замовляє → заклад підтверджує → клієнт отримує посилання на оплату. Підходить для залів.
Клієнт обирає сам
Обидва варіанти доступні. Клієнт бачить кнопки "Оплатити онлайн" та "Оплатити при отриманні".
Дозволити оплату готівкою / при отриманні
Клієнт може обрати "Оплата при отриманні" замість онлайн-оплати
